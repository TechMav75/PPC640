(2) Step to the track specified in the symmetry item of Table 4.1.
(3) Set the oscilloscope as follows:

CH1 : RD
CH2 : T1

Set time base to 1ms/division and vertical deflection to 0.2 v/division.

(4) Write 1 F.
(5) A read wave form is displayed on the oscilloscope as shown in Fig. 5.7.
(6) Confirm if it satisfies the value as shown in Table 4.1.
